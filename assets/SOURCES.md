# Звуковые ресурсы

Все встроенные записи используются по CC0 / public-domain лицензии.

- `boxing-start.wav` — Boxing Bell, автор IOII-IOIO-IIOI, CC0: https://commons.wikimedia.org/wiki/File:Dingdingding.wav
- `boxing-rest.wav` — Boxing Match Bell, Umplix, CC0: https://opengameart.org/content/boxing-ring-0
- `clear-bell.wav` — Pleasing Bell, Spring Spring, CC0: https://opengameart.org/content/pleasing-bell-sound-effect
- `coach-whistle.wav` — Steam Whistle, bart, CC0: https://opengameart.org/content/steam-whistle
- `gong.wav` — Gong or Bell Vibrant (Short), PDSounds, CC0: https://commons.wikimedia.org/wiki/File:Gong_or_bell_vibrant_(short).ogg

Файлы с суффиксом `-loud.wav` — технически нормализованные копии соответствующих CC0‑записей для отчётливого звука на смартфоне.
